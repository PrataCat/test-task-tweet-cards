Додаток Tweet-cards розроблений на базі create-react-app. Має 2 сторінки з хедером. На домашній сторінці відображається вітання-посилання на другу сторінку.
На другій сторінці завантажуються картки з твітами. Використовуючи кнопку "Follow/Following", можна підписатися/відписатися від твіту.
Деплой виконано на gh-pages. Для відкриття живої сторінки достатньо перейти за посиланням: https://pratacat.github.io/test-task-tweet-cards/

Використаний стек технологій: CSS, Styled-components, Axios, JC, React(у тому числі React Icons, React-content-loader, React Router), GitHub, VS Code.
